## Tips

### GHCI

- Load source file

```sh
:l[oad] demo.hs
```

- View type of a symbol

```sh
:t[ype] ($)
```

- View information of a symbol

```sh
:i[nfo] print

:i[info] Applicative
```

- View documentation of a symbol

```sh
:doc Applicative
```
