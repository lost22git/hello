## Resources

- [fullmoon](https://github.com/pkulchenko/fullmoon)

## Usage

Initialize fullmoon project

```sh
just fm-init
```

Build fullmoon project

```sh
just fm-build
```

Run fullmoon project

```sh
just fm-run
```

Run testing

```sh
just fm-test
```
