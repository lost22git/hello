# Odin Lang Learning

## Installation 

https://odin-lang.org/docs/install

### Debian

- Install llvm

https://apt.llvm.org

```shell
apt install llvm-<version>-dev clang-<version> -y
ln -sf /usr/bin/clang-<version> /usr/bin/clang
```

- Clone Odin source and build it

```shell
git clone git@github.com:odin-lang/Odin.git && cd Odin
make release-native
```
