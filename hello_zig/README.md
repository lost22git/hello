# Zig Lang Learning

## Installation

https://github.com/ziglang/zig/wiki/Building-Zig-From-Source

```shell
git clone git@github.com:ziglang/zig.git && cd zig
mkdir build
cd build
cmake ..
make install
ln -sf $(pwd)/stage3/bin/zig /usr/local/bin/zig
zig fmt
```
